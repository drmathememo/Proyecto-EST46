
Lentes 46 — PWA lista para publicar
===================================

Contenido del ZIP:
- index.html  → archivo único con toda la app PWA
- README.txt  → este instructivo paso a paso

Publicación 100% gratuita (elige 1):

─────────────────────────────────────────────────────────
Opción A) GitHub Pages (gratis y estable)
─────────────────────────────────────────────────────────
1) Crea una cuenta en https://github.com (si no tienes).
2) Arriba a la izquierda, clic “New” (nuevo repositorio).
   • Repo name: proyecto-est46 (o el nombre que quieras)
   • Visibilidad: Public
   • Clic “Create repository”.
3) En la página del repo, suelta (drag & drop) este archivo: index.html
   y luego baja y presiona “Commit changes”.
4) Ve a: Settings → Pages.
   • En “Build and deployment” → Source: “Deploy from a branch”
   • Branch: “main” y “/ (root)” → “Save”.
5) Espera ~1–2 minutos. Verás un link como:
   https://TU-USUARIO.github.io/proyecto-est46/
6) En iPhone: abre el link en Safari → botón “Compartir” → “Añadir a pantalla de inicio”.

─────────────────────────────────────────────────────────
Opción B) Netlify (gratis con plan básico)
─────────────────────────────────────────────────────────
1) Entra a https://app.netlify.com/ y accede con Google/GitHub.
2) “Add new site” → “Deploy manually”.
3) Arrastra este archivo index.html (o todo el ZIP).
4) Copia la URL pública (algo como https://xxxx.netlify.app).
5) En iPhone (Safari): abre la URL → “Compartir” → “Añadir a pantalla de inicio”.

─────────────────────────────────────────────────────────
Opción C) Google Drive como web simple
─────────────────────────────────────────────────────────
1) Sube index.html a tu Google Drive.
2) Clic derecho → “Compartir” → “Cualquiera con el enlace”.
3) Usa un publicador estático (por ejemplo https://gdriv.es/) para servirlo como página.
4) Abre el link resultante en Safari → “Añadir a pantalla de inicio”.

─────────────────────────────────────────────────────────
Consejos
─────────────────────────────────────────────────────────
• Para actualizar la app: reemplaza index.html en GitHub/Netlify.
• En iPhone: abre el sitio en Safari y desliza hacia abajo para refrescar la caché.
• Si usas Webhook, pégalo en el campo dentro de la app (se guarda en el celular).
